# UDA_CourseProject
Курсовой проект по анализу неструктурированных данных, в ходе которого были использованы различные BERT модели (fine-tuning) с целью предсказать наличия сарказма в заголовках статей(https://www.kaggle.com/rmisra/news-headlines-dataset-for-sarcasm-detection). Также была проведена визуализация данных с использованием WordClouds и Tfidf.
